SAC Consolidation Parser v1.0

Disclaimer: This program does not solve problems regarding duplicate barcodes. On the other hand, the creation of this program aims to solve problems that were frequently occurring during the uploading process.

Program Flow For Sanitizing:

1. First, fill-out the province and municipality PSG code and select the CSV file to be sanitized. Then run perform sanitize. Please check the downloaded csv file for any needed manual checking.

Program Flow For Xlsx to CSV Conversion:

1. First, erase previously stored files the converter > files > For_Conversion folder, then copy xlsx files for conversion in the aforesaid directory. Then, run the Convert xlsx to csv.
2. Second, check the converted files at converter > files > Processed_Excel folder. Take note, before execution of the conversion algorithm, files in the For_Conversion and Processed_Excel folders were automatically deleted.
